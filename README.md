# 🚀 AI Resume Analyzer SaaS

A smart ATS (Applicant Tracking System) Resume Analyzer that evaluates resumes, provides a score, and suggests improvements.

---

## 🔥 Features
- 📄 PDF Resume Upload (PDF.js)
- 🧠 Keyword Extraction & Skill Matching
- 📊 ATS Score Calculation (Weighted Algorithm)
- 📉 Missing Skills Detection
- 📈 Data Visualization (Chart.js)
- 👤 User Dashboard (LocalStorage)
- 🕘 Resume History Tracking

---

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Chart.js
- PDF.js
- LocalStorage

---

## ⚙️ How It Works
1. Upload or paste resume
2. Extract text from PDF
3. Match skills & keywords
4. Calculate ATS score
5. Display suggestions + chart
6. Save history
